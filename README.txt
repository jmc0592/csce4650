Type "make clean" into the terminal to ensure old files are deleted before every "make" command. After, type "make" into the terminal and hit Enter. 
This should create the 'compiler' executable file. You can then test it with various files by running './compiler < testFiles/fileName' where ‘fileName’ is any file within the testFiles directory.

The program will output what it sees when running through the lexor, and what actions are taken when in the parser.
